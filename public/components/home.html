Example page for using 'kondiskalkulatoren' API

This page is meant for giving an example how to use the API for 'kondiskalkulatoren'. You are advised to both play around on this, page but more importantly, have a look at the source code.

Documentation for the API is found here

If you have any questions, please send an email to 'haraldlons@gmail.com'
